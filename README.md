# NestJS Realtime Chat - Backend

This repository contains the backend of a real-time chat application, build with NestJS and Fastify, using WebSockets (Socket.IO) for real-time communication between clients.

## Project Goal

The goal of this project is to implement a real-time chat backend, focusing on:

- Modular architecture with NestJS
- WebSocket-based communication
- Connections and event management
- Clean separation between gateway and services
- Foundation for authentication and scalability
